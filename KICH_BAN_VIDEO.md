# Kịch bản video — Taoden IMEI Tool (~3 phút)

**Định dạng:** Quay màn hình Mac, có thể quay **im lặng** rồi đọc lời thoại bằng AI (Tutorial AI / CapCut / ElevenLabs).

**Chuẩn bị trước khi quay:**
- iPhone đã sẵn sàng + cáp USB
- Ảnh chụp màn hình **Cài đặt → Giới thiệu** (IMEI giả, không dùng máy khách thật)
- App đã build hoặc chạy `python main.py`
- Bảng trống hoặc xóa dữ liệu demo cũ

**Tốc độ đọc:** ~150 từ/phút → toàn bộ khoảng **450–480 từ**.

## Cảnh 1 — Mở đầu (0:00 – 0:20)

| | |
|---|---|
| **Màn hình** | Logo / icon app → double-click mở **Taoden IMEI Tool**. Nếu Gatekeeper chặn: chuột phải → **Open** → **Open**. |
| **Lời thoại** | «Xin chào! Đây là hướng dẫn nhanh **Taoden IMEI Tool** — phần mềm giúp cửa hàng đọc IMEI, Serial, Model iPhone qua cáp USB hoặc ảnh chụp màn hình, rồi xuất Excel và in nhãn.» |

## Cảnh 2 — Giao diện (0:20 – 0:35)

| | |
|---|---|
| **Màn hình** | Pan chậm: menu **Tệp · Nhập liệu · Bảng · Cài đặt** → bảng giữa → panel **Chi tiết dòng chọn** bên phải → thanh trạng thái dưới cùng. |
| **Lời thoại** | «Giao diện gồm bảng danh sách máy ở giữa, chi tiết từng dòng bên phải, và menu thao tác phía trên. Dữ liệu tự lưu — tắt app vẫn còn.» |

---

## Cảnh 3 — Đọc qua USB (0:35 – 1:10)

| | |
|---|---|
| **Màn hình** | ① Cắm iPhone vào Mac. ② Trên iPhone: bấm **Tin cậy**. ③ Quay lại app — đợi vài giây — dòng mới xuất hiện, **Nguồn: USB**. ④ Click dòng → panel phải hiện IMEI, Serial, Model, iOS, Màu, % Pin. |
| **Lời thoại** | «Cách nhanh nhất: cắm iPhone bằng cáp, mở khóa máy và bấm **Tin cậy** máy tính. App tự đọc IMEI, Serial, Model, phiên bản iOS, màu máy, dung lượng và phần trăm pin. Không cần bấm nút gì thêm. Cắm lại cùng máy thì app cập nhật dòng cũ, không tạo trùng.» |

---

## Cảnh 4 — Nhập từ ảnh OCR (1:10 – 1:45)

| | |
|---|---|
| **Màn hình** | ① **Nhập liệu → Đọc từ ảnh** (hoặc ⌘I). ② **Chọn ảnh** màn hình Giới thiệu iPhone — hoặc dán ảnh ⌘V. ③ App nhận diện IMEI, Serial, Model. ④ **Thêm vào bảng**. |
| **Lời thoại** | «Không có cáp? Chụp màn hình **Cài đặt → Cài đặt chung → Giới thiệu** trên iPhone. Vào menu **Nhập liệu**, chọn **Đọc từ ảnh**, tải ảnh hoặc dán trực tiếp. App dùng OCR để đọc IMEI và Serial, rồi bấm **Thêm vào bảng**. Nên crop sát vùng Giới thiệu cho chính xác.» |

---

## Cảnh 5 — Sửa & tick chọn (1:45 – 2:00)

| | |
|---|---|
| **Màn hình** | ① Click ô **Hình thức** hoặc **Model** → sửa → Enter. ② Click cột ☐ tick 2–3 dòng. ③ Click tiêu đề cột ☐ → chọn tất cả. |
| **Lời thoại** | «Click vào ô bất kỳ để sửa trực tiếp, Enter để lưu. Cột tick đầu dòng dùng để chọn máy cần in hoặc xóa. Click tiêu đề cột tick để chọn tất cả.» |

---

## Cảnh 6 — In nhãn (2:00 – 2:25)

| | |
|---|---|
| **Màn hình** | ① Tick dòng cần in. ② **Tệp → In đã tick** (⌘P). ③ Preview mở file PDF — zoom nhãn: IMEI, Model, Màu, barcode. ④ ⌘P in ra máy in (hoặc chỉ show PDF). |
| **Lời thoại** | «Tick chọn máy cần in, vào **Tệp → In đã tick**. App tạo PDF nhãn — mỗi máy một trang, có mã vạch quét ra IMEI. Mở Preview và in như file PDF bình thường.» |

---

## Cảnh 7 — Xuất Excel & Cài đặt (2:25 – 2:45)

| | |
|---|---|
| **Màn hình** | ① **Tệp → Xuất Excel** — file mở tự động. ② **Cài đặt → Cài đặt** — lướt tick/bỏ tick vài cột bảng và mục in → **Lưu** → bảng thay đổi cột hiển thị. |
| **Lời thoại** | «**Xuất Excel** để gửi danh sách toàn bộ máy. Trong **Cài đặt**, bạn chọn cột nào hiện trên bảng, và mục nào in trên nhãn — IMEI, Model, Màu, Dung lượng, Pin, iOS hay Barcode.» |
